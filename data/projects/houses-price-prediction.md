# Houses Price Prediction

A machine learning project for predicting house sale prices on a structured
dataset.

## Overview

Built a machine learning model to predict sales prices for houses within a
specific dataset using various ML algorithms.

## Highlights

- Data exploration, cleaning, and feature engineering on the housing dataset.
- Comparison between several regression algorithms to identify the best
  performing model.
- Evaluation using standard regression metrics (RMSE / MAE / R²).
- End-to-end pipeline from raw data to trained model.
