# Assets

Place your profile image here as **`profile.jpg`** (or `profile.png` and update
the references in `index.html` / `pages/about.html`).

Recommended:
- Format: JPG or PNG
- Aspect ratio: 4:5 (portrait) for best results in the hero section
- Size: at least 600 × 750 px
- File name: `profile.jpg`

If the image is missing the portrait frame will gracefully show a placeholder
label so the layout stays intact.
